# Hangman Game (Python)

## Description
This is a simple Hangman game developed using Python. The player has to guess a hidden word by entering one letter at a time within limited attempts.

## Features
- Random word selection
- User input validation
- Tracks guessed letters
- Limited attempts
- Displays win or loss result

## How It Works
- A random word is selected from a list
- The word is displayed as underscores (_ _ _)
- The player guesses one letter at a time
- If the guess is correct, the letter is revealed
- If the guess is wrong, attempts decrease
- The game ends when the word is guessed or attempts run out
